💬 Convyx – Real-Time Chat Application

Convyx is a real-time one-to-one chat application inspired by modern messaging platforms like WhatsApp. It is built using the MERN stack and Socket.IO, focusing on real-time communication, media sharing, and a smooth user experience.

🚀 Features

📱 OTP-based Authentication using mobile number

💬 Real-time private messaging with Socket.IO

✍️ Typing indicators

👀 Read receipts (sent, delivered, seen)

😀 Emoji reactions to messages

↩️ Reply & forward messages

🗑️ Delete messages (for me / everyone)

🖼️ Image & video sharing

🎤 Voice message support

📂 Contact-based chat visibility (users see only their contacts)

🔒 Secure authentication using JWT

🛠️ Tech Stack
Frontend

React.js

Context API

Axios

Socket.IO Client

CSS (custom UI)

Backend

Node.js

Express.js

MongoDB

Mongoose

Socket.IO

Multer (media & audio uploads)

JWT Authentication

⚙️ Application Architecture

REST APIs are used for authentication, user management, and media uploads

Socket.IO handles real-time features such as:

Message delivery

Typing indicators

Reactions

Read receipts

Online presence

🔑 Authentication Flow

User registers using a mobile number

OTP is sent and verified (demo/local setup)
